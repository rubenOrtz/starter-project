# Reporte de Desarrollo - Applicant Showcase App

## 1. Introducción

**Preambulo:**
Al recibir el desafío de Symmetry, mi enfoque principal ha sido no solo cumplir con los requisitos
funcionales, sino demostrar una comprensión profunda de la arquitectura de software escalable.
Quieor demostrar lo que valgo.

**Objetivo del Proyecto:**
El propósito principal de este desarrollo fue implementar la funcionalidad de "Periodista" en la App
existente. Permitiendo al perodista subir sus propios artículos, integrando una solución completa
que abarca desde la interfaz de usuario (Flutter) hasta el backend (Firebase), todo bajo una
estricta Clean Architecture.
